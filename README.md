# Laravel based Webshop
## Intro
Laravel based Portfolio application.

## Application
This application has the following:
* Authentication
  * Login to access CMS
  * Except:
    * Registration
    * Recovery
    * Reset
* CMS
  * Profile
    * CRUD
    * Except: Create, Delete
  * Education
    * CRUD
  * Employment
    * CRUD
  * Skills
    * CRUD
  * Softskills
    * CRUD
  * Projects
    * CRUD
* File upload
  * Laravel Storage
* Frontend
  * Bootstrap Template
